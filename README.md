# Front-end Cielo

> Repositório Back-end do Desafio Front-end proposto pela Cielo

## Stack Utilizada

![node-image] <small >v14.15.4 </small>

## Instalação Back-End

execute o comando abaixo.

```sh
npm install
```

## Executando a Aplicação

Após realizar a instalação, execute o comando abaixo

```sh
npm start
```
